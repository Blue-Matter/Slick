# Assign or access `TimeTerminal` for a valid object class

Assign or access `TimeTerminal` for a valid object class

## Usage

``` r
TimeTerminal(object)

TimeTerminal(object) <- value

# S4 method for class 'Kobe'
TimeTerminal(object)

# S4 method for class 'Kobe'
TimeTerminal(object) <- value
```

## Arguments

- object:

  An object of class
  [`Kobe-class()`](https://slick.bluematterscience.com/reference/Kobe-class.md)

- value:

  A `numeric` value with a value matching one in `Kobe@Time`
  [`Kobe-class()`](https://slick.bluematterscience.com/reference/Kobe-class.md)
  for details.

## Value

Returns a numeric values from the `TimeTerminal` slot in
[`Kobe()`](https://slick.bluematterscience.com/reference/Kobe-methods.md)
objects

## Methods (by class)

- `TimeTerminal(Kobe)`: Return `TimeTerminal` from a
  [`Kobe-class()`](https://slick.bluematterscience.com/reference/Kobe-class.md)
  object

- `TimeTerminal(Kobe) <- value`: Assign `TimeTerminal` to a
  [`Kobe-class()`](https://slick.bluematterscience.com/reference/Kobe-class.md)
  object
