# Access and return `TimeNow` from a [`Timeseries-class()`](https://slick.bluematterscience.com/reference/Timeseries-class.md) object

Access and return `TimeNow` from a
[`Timeseries-class()`](https://slick.bluematterscience.com/reference/Timeseries-class.md)
object

## Usage

``` r
TimeNow(Timeseries)

TimeNow(Timeseries) <- value
```

## Arguments

- Timeseries:

  A
  [`Timeseries-class()`](https://slick.bluematterscience.com/reference/Timeseries-class.md)
  object

- value:

  A character string label for the time axis. Use a named list for
  multiple languages

## Value

A numeric value

## Functions

- `TimeNow(Timeseries) <- value`: Assign `TimeNow` to a
  [`Timeseries-class()`](https://slick.bluematterscience.com/reference/Timeseries-class.md)
  object
