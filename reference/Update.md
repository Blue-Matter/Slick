# Updates an old object of class `Slick` to new S4 class `Slick`

Updates an old object of class `Slick` to new S4 class `Slick`

## Usage

``` r
Update(slick)
```

## Arguments

- slick:

  An S3 object of class `Slick`

## Value

An S4 object of class `Slick`

## Examples

``` r
slick <- Slick() # a dummy old Slick object
slick <- Update(slick) # updated to S4 object
```
