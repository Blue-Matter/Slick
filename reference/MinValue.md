# Assign and access `MinValue` and `MaxValue` for a `Quilt` object

Assign and access `MinValue` and `MaxValue` for a `Quilt` object

## Usage

``` r
MinValue(Quilt)

MinValue(Quilt) <- value

MaxValue(Quilt)

MaxValue(Quilt) <- value
```

## Arguments

- Quilt:

  A
  [`Quilt-class()`](https://slick.bluematterscience.com/reference/Quilt-class.md)
  object

- value:

  A numeric vector with minimum or maximum values for the Performance
  Indicators.

## Value

A numeric value

## Functions

- `MinValue()`: Return `MinValue` from a
  [`Quilt-class()`](https://slick.bluematterscience.com/reference/Quilt-class.md)
  object

- `MinValue(Quilt) <- value`: Assign `MinValue` to a
  [`Quilt-class()`](https://slick.bluematterscience.com/reference/Quilt-class.md)
  object

- `MaxValue()`: Return `MaxValue` from a
  [`Quilt-class()`](https://slick.bluematterscience.com/reference/Quilt-class.md)
  object

- `MaxValue(Quilt) <- value`: Assign `MaxValue` to a
  [`Quilt-class()`](https://slick.bluematterscience.com/reference/Quilt-class.md)
  object
